## 🏗️ Architecture Overview

### Three-Tier Architecture
1. **CLI Layer** (`fpcr.py`, `cpsearch.py`) - Command-line interface
2. **Business Logic** (`cpcrud/`, `fa/`) - CRUD processing, FastAPI app
3. **Data Layer** (`libs/cpaiops/`) - Check Point API integration

### Key Patterns
- **Service-Adapter Pattern**: CPAIOPS library uses ports/adapters
- **Repository Pattern**: Cache repositories for data access
- **Orchestration Layer**: Smart caching with session tracking
- **YAML Templates**: CPCRUD uses YAML for rule templates

### Dependencies
- **cpaiops** - Internal Check Point API library
- **arlogi** - Internal logging
- **typer** - CLI framework
- **rich** - Terminal formatting
- **fastapi** - Web UI backend
- **react** - Web UI frontend
